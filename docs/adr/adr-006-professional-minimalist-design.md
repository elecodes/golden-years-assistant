# ADR-006: Professional Minimalist Design System

**Status**: Accepted

**Date**: 2024-03-18

## Context

The original UI design appeared too childish with bright blue backgrounds and heavy blue borders. For a healthcare/elderly care application, we needed a more professional, mature aesthetic that inspires trust while maintaining accessibility.

## Decision

We adopted a professional minimalist design system:

- **Primary**: Deep navy (#0F172A) for headers and primary text
- **CTA/Accent**: Sky blue (#0369A1) for actions and links
- **Success**: Emerald (#059669) for completed states
- **Warning**: Amber (#D97706) for low-stock alerts
- **Emergency**: Red (#DC2626) for emergency button
- **Background**: Light slate (#F8FAFC) for page background

### Design Principles

- Clean, flat cards with subtle borders (1px)
- Subtle shadows for depth
- Semibold typography with tight tracking
- Smaller, refined iconography
- Professional color contrast

## Consequences

### Positive
- Professional appearance that inspires trust
- Mature aesthetic appropriate for healthcare
- Clean hierarchy with refined spacing
- Maintains accessibility standards

### Negative
- Less "friendly" visual approach
- May need further refinement based on user feedback

## Alternatives Considered

### Previous Design (Bright Blue)
- Too childish for healthcare
- Overwhelming visual weight
- Harder to read for elderly users

### High Contrast Dark Mode
- Too aggressive for elderly users
- May cause eye strain
- Not appropriate for healthcare context
